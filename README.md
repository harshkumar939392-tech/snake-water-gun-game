# 🐍 Snake Water Gun Game

A simple command-line **Snake Water Gun Game** developed using **Python**. This game is based on the classic Snake-Water-Gun rules, where the user plays against the computer.

---

## 🎮 Game Rules

- 🐍 Snake drinks Water → **Snake wins**
- 💧 Water douses Gun → **Water wins**
- 🔫 Gun kills Snake → **Gun wins**
- Same choice by both players → **Draw**
